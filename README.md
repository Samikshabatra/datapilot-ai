# 📊 DataPilot AI

> Your Copilot for Intelligent Data Exploration

Upload datasets, ask questions in natural language, and receive automated exploratory data analysis (EDA), visualizations, insights, and executable Python workflows.

## 🔗 Links

* 🚀 Live Demo: https://huggingface.co/spaces/samikshabatra18/datapilot-ai
* 💻 Repository: https://github.com/Samikshabatra/datapilot-ai

## ✨ Features

* 📂 Upload CSV and Excel datasets
* 💬 Conversational querying using natural language
* 📈 Automated Exploratory Data Analysis (EDA)
* 📊 Interactive visualizations and charts
* 🐍 AI-generated executable Python code
* 🔄 Self-correcting code execution pipeline
* 📝 Human-readable explanations of findings
* 🚀 Dockerized deployment on Hugging Face Spaces

## 🏗️ Architecture

User Upload → Dataset Profiling → LLM Agent → Code Generation → Sandbox Execution → Visualization → Insight Synthesis
